# Themes Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a 4-theme system (Default, Dark, Light, Claude Code) with a dropdown switcher.

**Architecture:** CSS custom properties on `[data-theme="X"]` selectors, mapped through Tailwind v4 `@theme inline` so components reference semantic tokens (`bg-surface`, `text-accent`, etc.). Theme state stored in zustand (persisted). A `ThemeProvider` client component sets `data-theme` on `<html>`.

**Tech Stack:** Tailwind v4 CSS variables, zustand v5, Next.js 16 App Router

### Task 3: ThemeProvider Client Component

**Files:**
- Modify: `app/layout.tsx`

- [ ] **Step 1: Add ThemeProvider component and mount it in the layout**

Create an inline client component in `layout.tsx` (or a separate file — inline is fine since it's tiny). It reads `theme` from zustand and sets `data-theme` on `<html>`.

Add this before the `export default function RootLayout`:

```tsx
'use client';

import { useEffect } from 'react';
import { useChatStore } from './store/chatStore';

function ThemeProvider({ children }: { children: React.ReactNode }) {
  const theme = useChatStore((s) => s.theme);

  useEffect(() => {
    document.documentElement.setAttribute('data-theme', theme);
  }, [theme]);

  // On mount, respect prefers-color-scheme unless a theme was already persisted
  useEffect(() => {
    const stored = JSON.parse(localStorage.getItem('blues-chat-storage') || '{}');
    if (!stored?.state?.theme || stored.state.theme === 'default') {
      const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
      const osTheme = prefersDark ? 'dark' : 'light';
      // Only apply OS default if no explicit theme choice was made
      document.documentElement.setAttribute('data-theme', osTheme);
    }
  }, []);

  return <>{children}</>;
}
```

Wrap children in layout:

```tsx
<body className="min-h-full flex flex-col">
  <ThemeProvider>{children}</ThemeProvider>
</body>
```

Note: Since layout.tsx is a server component, you need to make ThemeProvider a separate client component file or keep the `'use client'` directive contained. Best approach: create `app/components/ThemeProvider.tsx` with the above code, import it in layout.
